Characteristics of a PWA

  • Responsive
  • Connectivity independent
  • App-like interactions
  • Fresh
  • Safe
  • Discoverable
  • Re-engageable
  • Installable
  • Linkable

To be considered a PWA:

  • Secure - SSL
  • Available Offline
  • Manifest
  • Icon

  • Better web applications
  • Broader audience
  • Application store friction
  • Multiple platforms

Not convinced?

  • 65% of users install 0 apps a month
  • 84% of users spend time in 5 apps
  • 95% for tablets!
  • Web sites have 4.5x greater reach
